# QR Code Detector

#### 关键参数
> 关键参数请在 **qrcode_detector.cpp** 中的 **QRCodeDetector::QRCodeDetector** 中修改  
> **show_result_** 是否显示结果，可以通过 **setShowResult（）** 重新设计  
> **QRCodeInfo** ,**type_name**:类型，二维码还是条形码；**data**:数据，二维码携带的数据;**vertex_position**:二维码四个顶点,0-3从左上角逆时针排列  

#### 函数说明
> **qrcode_detector.h**  
> **detector** 识别函数
